## Subject 1: Internet of Things (IoT)

**Q1. What is the definition of IoT?**
Ans: The Internet of Things (IoT) refers to a network of physical devices, vehicles, appliances, and other objects embedded with sensors, software, and network connectivity, allowing them to collect and exchange data. It enables seamless communication between people, processes, and things.

**Q2. List some key characteristics of IoT.**
Ans: Key characteristics include dynamic and self-adapting behavior, self-configuring capability, interoperability via standard communication protocols, unique identities for each device, and integration into the information network.

**Q3. Describe the IoT Conceptual Framework.**
Ans: The conceptual framework categorizes IoT into elements like gather (sensors collect data), enrich (enriching data context), stream (transmitting data securely), manage (managing devices and analytics), and acquire (deriving insights and acting on data).

**Q4. Explain the IoT Architectural View.**
Ans: The architectural view consists of multiple layers: the Device/Perception layer (sensors/actuators), the Network layer (data transmission), the Middleware layer (data processing and storage), and the Application layer (delivering services to users).

**Q5. What constitutes the Physical Design of IoT?**
Ans: The physical design describes the tangible elements of IoT, conceptualized as 'Things' (devices with interfaces for sensing/actuation and networking) and the associated IoT protocols (link, network, transport, and application layers like Wi-Fi, IPv4, TCP, HTTP).

## Subject 2: Artificial Intelligence and Machine Learning

**Q1. What is Artificial Intelligence?**
Ans: AI is a branch of computer science emphasizing the creation of intelligent machines that work, learn, and react like humans. It involves designing systems capable of problem-solving, learning, speech recognition, and planning.

**Q2. What are Production Systems in AI?**
Ans: A production system is a cognitive mechanism consisting of a set of rules (condition-action pairs), a working memory that stores the current state, and a control strategy (recognize-act cycle) that dictates how rules are applied to the working memory.

**Q3. State the characteristics of Production Systems.**
Ans: They are characterized by simplicity, modularity, and knowledge-intensive nature. They separate knowledge (rules) from control (engine), allowing easy rule updates. However, they can suffer from inefficiency due to the massive search space in complex problems.

**Q4. Explain Breadth-First Search (BFS).**
Ans: BFS is an uninformed search strategy that explores a graph/tree by expanding all nodes at the present depth level before moving on to nodes at the next depth level. It uses a FIFO queue and guarantees finding the shortest path in an unweighted graph.

**Q5. Explain Depth-First Search (DFS).**
Ans: DFS is an uninformed search strategy that explores as far as possible along each branch before backtracking. It uses a LIFO stack. It is memory efficient compared to BFS but is not guaranteed to find the optimal solution and can get stuck in infinite loops.

## Subject 3: Compiler Design

**Q1. What is a Compiler?**
Ans: A compiler is a system software that translates code written in a high-level programming language (source language) into an equivalent low-level language (target language, such as machine code or assembly), reporting errors encountered during the process.

**Q2. Briefly explain the major data structures in a compiler.**
Ans: The primary data structures include tokens (output of the lexer), syntax trees (hierarchical representation of the code structure), symbol tables (storing identifiers and their attributes), intermediate representations (like DAGs or three-address code), and various error logs.

**Q3. What are the types of Compilers?**
Ans: Compilers can be categorized as Cross-compilers (generating code for a platform different from the one it runs on), Source-to-source compilers (translating between high-level languages), Just-In-Time (JIT) compilers (compiling at runtime), and single-pass/multi-pass compilers.

**Q4. Differentiate between the Front-end and Back-end of a Compiler.**
Ans: The front-end focuses on analyzing the source code (lexical, syntax, and semantic analysis) and generating intermediate representation. It is language-dependent. The back-end focuses on optimizing the intermediate code and generating the target machine code. It is machine-dependent.

**Q5. What is the Analysis-Synthesis model of compilation?**
Ans: Compilation is divided into two phases. The Analysis phase breaks the source program into constituent pieces and creates an intermediate representation, dealing with syntax and semantics. The Synthesis phase constructs the desired target program from this intermediate representation.

## Subject 4: Natural Language Processing

**Q1. What is Natural Language Understanding (NLU)?**
Ans: NLU is a subtopic of NLP that focuses on machine reading comprehension. It attempts to deduce the intended meaning of human language, handling nuances, context, and intent to enable machines to understand text similarly to humans.

**Q2. List the levels of language analysis in NLP.**
Ans: The levels include Phonology (sounds), Morphology (word formation), Lexical (word meaning), Syntactic (sentence structure), Semantic (sentence meaning), Pragmatic (contextual meaning), and Discourse (meaning across sentences).

**Q3. Explain Syntax and Semantics in NLP.**
Ans: Syntax refers to the grammatical rules that govern how words are arranged to form sentences, ensuring structural correctness. Semantics deals with the actual meaning conveyed by the words and sentences, ensuring logical correctness regardless of structure.

**Q4. What is Pragmatics in NLP?**
Ans: Pragmatics is the study of how context influences the interpretation of meaning. It goes beyond the literal meaning of sentences (semantics) to consider the speaker's intent, the situation, and world knowledge to resolve ambiguities and understand implied meanings.

**Q5. Mention some applications of NLP.**
Ans: Major applications include machine translation (e.g., Google Translate), sentiment analysis (classifying text polarity), chatbots and virtual assistants (e.g., Siri, Alexa), text summarization, speech recognition, and information extraction.

## Subject 5: Scientific Aptitude

**Q1. If a nine-digit Number 985x3678y is divisible by 72, then the Value of (4x-3y)?**
Ans: For divisibility by 72, it must be divisible by 8 and 9. For 8, the last 3 digits '78y' must be divisible by 8. Thus, y=4 (784/8 = 98). The sum of digits = 9+8+5+x+3+6+7+8+4 = 50+x. For divisibility by 9, 50+x must be divisible by 9, so x=4. Therefore, (4x - 3y) = 4(4) - 3(4) = 16 - 12 = 4.

**Q2. In a 7 digit number 89476*2, what is the smallest possible value of “* “such that the number is divisible by 8?**
Ans: For divisibility by 8, the last three digits must be divisible by 8. The last three digits are '6*2'. Let's test *: 612/8 = 76.5. 632/8 = 79. So the smallest possible value of * is 3.

**Q3. Find the Difference between Squares of the greatest value and the smallest value of P if the number 5306P2 is divisible by 3?**
Ans: For divisibility by 3, the sum of digits must be a multiple of 3. Sum = 5+3+0+6+P+2 = 16+P. P can be 2, 5, or 8. Greatest value is 8, smallest is 2. Difference between their squares = 8^2 - 2^2 = 64 - 4 = 60. 

**Q4. If the sales tax be reduced from 3.5% to 3.33% then what difference does it make to a person who purchases on article with market price of 8400?**
Ans: Tax difference = 3.5% - 3.333% = 0.1666% (which is 1/6 %). Difference in residential amount = (1/600) * 8400 = 14.

**Q5. If X% of Y is 150 and Y% of Z is 300, then find the Relation between X and Z?**
Ans: (X/100)*Y = 150 => XY = 15000. (Y/100)*Z = 300 => YZ = 30000. Dividing the two equations gives (XY)/(YZ) = 15000/30000 => X/Z = 1/2. Thus, Z = 2X.
